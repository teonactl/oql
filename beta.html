⚠️ Istruzioni di installazione — leggi prima di aprire il file
🐧 Linux — AppImage
  1. Se il browser chiede "Mantenere il file?" clicca Mantieni.
  2. Rendi eseguibile il file e avvialo:
    chmod +x oql-beta-linux.AppImage && ./oql-beta-linux.AppImage
🍎 macOS — DMG
  1. Se il browser chiede "Consentire il download?" o "Mantenere?" clicca Consenti / Mantieni.
  2. Apri il file .dmg scaricato.
  3. Trascina OQL nella cartella Applicazioni.
  4. Espelli il disco e apri il Terminale (Applicazioni → Utility → Terminale).
  5. Esegui questo comando:
    xattr -cr /Applications/OQL.app
  6. Ora puoi aprire OQL normalmente dal Launchpad o dalla cartella Applicazioni.
Perché è necessario? OQL non è ancora notarizzata da Apple. macOS blocca qualsiasi app scaricata da internet non notarizzata, mostrando il messaggio "OQL è danneggiato e non può essere aperto". Il comando xattr -cr rimuove il flag di quarantena aggiunto automaticamente dal browser. Va ripetuto ad ogni aggiornamento.
🪟 Windows — Installer
  1. Se il browser mostra "Questo file potrebbe essere dannoso" clicca Mantieni comunque (o Keep anyway).
  2. Avvia il file .exe scaricato.
  3. Se appare Windows Defender SmartScreen con "Il PC è stato protetto":
    clicca "Ulteriori informazioni" → "Esegui comunque".
  4. Segui la procedura guidata di installazione.
Perché appare SmartScreen? L'installer OQL non è ancora firmato con un certificato code-signing a pagamento. SmartScreen avvisa per qualsiasi installer non firmato da un publisher noto. Il software è sicuro — il codice sorgente è pubblico su github.com/teonactl/oql.

I download beta vengono aggiornati manualmente. Se i link non funzionano la build beta non è ancora stata eseguita — verifica lo stato su GitHub Actions.
